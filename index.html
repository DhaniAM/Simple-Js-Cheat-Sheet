var vs let

Let can't be Redeclared, Var can be Redeclared


Doesn't work

let x = 10;
let x = 20;

Does work

var x = 10;
var x = 20;

Let has Block Scope, Var doesn't have Block Scope


let x = 5;
// x is 5 here

function() {
  let x = 8;
// x is 8 here
}

// x is 5 here
var x = 5;
// x is 5 here

function() {
  var x = 8;
// x is 8 here
}

// x is 8 here

Let must be Declared before use, Var can be use before Declaration (Hoisting)


This works

x = 10;
var x;

But these doesn't work

x = 10;
let x;

or

x = 10;
const x;

Object

Creating an Object

const person = {
  firstName: "John",
  lastName: "Doe",
  age: "30"
}

Accessing Object Properties

person.age //30

or

person["age"] //30

Changing Object Properties

person.age = 50;

or

person["age"] = 50

Object Methods

const person = {
  firstName: "John",
  lastName: "Doe",
  age: "30"
  function fullName() {
    return this.firstName + " " + this.lastName;
  }
};

let friendName = person.fullName(); // returning "John Doe"

Switch

let key = 10;
switch(key) {
  case 0:
    console.log("key is 0");
    break;
  case 5:
    console.log("key is 5");
    break;
  case 10:
    console.log("key is 10");
    break;
  case 15:
    console.log("key is 15");
    break;
  default:
    console.log("key is not found");
    break;
}
